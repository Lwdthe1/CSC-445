* Switching
* Internetworking
* IP and Routing
* TCP and UDP

* Congestion control and Quality of Service
* Application protocols: HTTP, FTP, etc
* Distributed Systems
* RPC and remote invocation
* Naming, Security, Caching
* Group communication
* Frameworks: CORBA, Jini, etc
